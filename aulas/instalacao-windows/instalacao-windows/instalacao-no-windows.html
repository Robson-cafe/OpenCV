Instalação no Windows

    • Faça download do Visual Studio no site(Versão community): https://visualstudio.microsoft.com/pt-br/ e escolha a opção [Desktop develompment with C++].
    • Faça o download do OpenCV: https://github.com/opencv/opencv em releases baixe o .exe : https://github.com/opencv/opencv/releases/download/4.5.3/opencv-4.5.3-vc14_vc15.exe e (Instale o 7zip e escolha a pasta C:/ ) dê um duplo clique e esoclha o local de instalação(Recomendação: C:). e clique em [Extract].
    • Verificar a localização da pasta bin do opencv, exemplo(vá até ela e dê Ctrl + L para poder copiar o caminho):

C:\opencv\build\x64\v15\bin

E adicionar à variável Path do Windows(No menu iniciar digite: variáveis de ambiente e abra em seguida ao encontrar, adicione)

    • Reinicie o computador
    • No Visual Studio CRIE UM NOVO PROJETO → Projeto Vazio(mude o nome para OpenCVcurso) → CREATE . Mude para x64!

    • Vá em Projeto → Propriedades → Diretórios VC++:

      1. [Diretório de Inclusão] C:\opencv\build\include
      1. [Diretórios de Biblioteca] C:\opencv\build\x64\vc15\lib

[Aplicar][Ok]

  1. De novo em: Projeto → Propriedades

E em Vinculador → Entrada → Dependências Adicionais (opencv_world450d.lib)

C:\opencv\build\x64\vc15\lib\opencv_world450d.lib

[Ok][Aplicar][Ok]

    • Crie um arquivo de teste do OpenCV e Compile:

Clicar com o botão direito do mouse sobre a pasta do projeto → Adicionar → Novo Item → (Nome Desejado do Arquivo.cpp) 

#include <opencv2/imgcodecs.hpp>
#include <opencv2/highgui.hpp>
#include <opencv2/imgproc.hpp>
#include <iostream>

int main(int argc, char** argv) {
    std::string caminho = "arquivos/minha-foto.png";
    cv::Mat imagem = cv::imread(caminho);
    cv::imshow("Imagem", imagem);
    cv::waitKey(0);
    return 0;
}

Clique com o botão direito do mouse sobre o projeto no Visual Studio e depois em Abrir diretório com gerenciador de arquivos. Crie uma pasta de nome [arquivos] e cole sua imagem(PNG) ou [essa] com o nome minha-foto.

Agora você precisa clicar em: [Depurador Local do Windows] vai dar erro, mas vai criar o diretório [Debug]

Agora você precisa COPIAR a DLL(opencv_world450d.dll) do caminho: C:\opencv\build\x64\vc15\bin para: C:\Users\Familia\Documents\OpenCVcurso\x64\Debug\

Agora novamente clique em [Depurador Local do Windows]. Se abrir uma janela com a imagem é porque está tudo certo, senão, verifique os passos .

Depois clique em [Continuar] para voltar a aparecer [Depurador Local do Windows].

Os próximos passos serão iguais ao Linux. Para criar arquivos para Programação Orientada à Objetos use outros tipos de arquivos(cabeçalho e source) quando for [Adicionar][Novo item] . Isso está dentro do escopo do uso do Visual Studio e não do OpenCV .
